# README

[[English version]](./README.md)

WFH 參加線上會議時，不論是 Google Meet 或是其他的會議軟體，  
有時想做筆記，但苦無簡報!! 
這時候你可能會想要截圖，但是節圖會充滿很多 ~~多餘的~~ 資訊。

本專案就是協助你擷取截圖後的畫面，讓你可以單純切出想要的簡報畫面。
